# Матрица ручных проверок ShiftAlarm

Чек-лист проверок на реальном устройстве перед сборкой/релизом. Автотесты покрывают движок и
формат данных; но срабатывание будильника, OEM-ограничения, экран звонка и системные события
проверяются только руками на телефоне.

Как пользоваться: перед релизом пройти по разделам, отмечать результат (✅/❌/—) и дату/устройство.
Для сменщика-ночника особенно важны разделы «Смены/ночные» и «Периоды».

Условные обозначения статуса: ✅ прошло · ❌ не прошло · — не проверяли.
